# Tags Generator Front

## Description

Tags generator front-end.
A Replit is deployed [here](https://replit.com/@lcdm-all/tag-gen-client#src/routes/index.svelte).

## Technical stack

- Node
- TypeScript
- Svelte
- Svelte Material UI

## Install

```bash
$ npm install

# Build the theme style file (cf svelte material UI foc)
$ npm prepare
```

## Running the app

```bash
# development
$ npm dev

# preview
$ npm run build
$ npm preview
```
